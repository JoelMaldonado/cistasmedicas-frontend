# Citas Médicas - Frontend

Frontend construido con [Vue 3](https://vuejs.org/) + [Vite](https://vitejs.dev/) + TypeScript.

## Requisitos

- Node.js 20+
- npm

## Configuración

Copia el archivo de variables de entorno de ejemplo y ajusta los valores según tu entorno:

```bash
cp .env.example .env
```

## Instalación

```bash
npm install
```

## Ejecutar en desarrollo

```bash
npm run dev
```

## Build de producción

```bash
npm run build
```
